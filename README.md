# Portafolio Personal — Agustina

## Descripción del Proyecto

Este es mi portafolio personal desarrollado como **Práctica Formativa Obligatoria 1 (PFO1)** de la materia de Desarrollo Web en el IFTS N°29. Se trata de una landing page estática construida únicamente con HTML5 y CSS3, sin frameworks ni librerías JavaScript externas. El diseño adopta un estilo dark mode con tipografía de Google Fonts y un layout responsivo que se adapta a distintos tamaños de pantalla.
